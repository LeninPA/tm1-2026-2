# Regresión lineal

Puedes consultar en el siguiente [enlace](https://sistemas.fciencias.unam.mx/~erhc/calculo3_20171/derivadas_parciales_direccionales_2016_12.pdf) el desarrollo detrás de las ecuaciones del mínimo de métodos cuadrados, junto con una demsotración de su optimalidad.

## Requisitos de instalación
Para ejecutar este código se requiere de instalar:

- pandas
- plotnine

Que usaremos para la visualización de los datos.
